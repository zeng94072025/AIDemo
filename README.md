# AI圖片優化助手

一個功能強大的AI圖片處理工具，支持多種圖片優化功能，包括智能修復、濾鏡特效、塗鴉標註等。

## 🚀 主要功能

### 圖片處理功能
- **智能圖片修復**：一鍵修復老照片、去除瑕疵
- **AI智能處理**：8K超分辨率、智能摳圖、背景替換
- **濾鏡特效**：豐富的濾鏡和特效庫
- **塗鴉標註**：直觀的塗鴉和標註工具
- **批量處理**：支持多張圖片同時處理
- **多格式支持**：JPG、PNG、WebP等主流格式

### 用戶認證系統
- **多種登錄方式**：
  - 密碼登錄（強制密碼強度驗證）
  - 短信驗證碼登錄（60秒動態碼 + 圖形驗證碼）
  - 微信/支付寶一鍵登錄
- **安全防護**：
  - 輸錯5次自動鎖定賬戶（15分鐘）
  - 圖形驗證碼防刷機制
  - 密碼強度實時檢測
  - 銀行級安全標準

### 支付系統
- **極簡支付流程**：3步完成支付，失敗率低於0.2%
- **多種支付方式**：微信支付、支付寶
- **金融級可靠性**：
  - 防超額支付（分布式鎖控制）
  - 防丟單機制（15分鐘輪詢補漏）
  - 支付簽名驗證（SHA256+RSA）
  - 5分鐘支付超時保護
- **實時狀態反饋**：支付成功/失敗即時通知

### 管理員儀表板
- **數據驅動決策**：
  - 實時收入統計（微信 vs 支付寶占比）
  - 用戶增長趨勢分析
  - 轉化率監控
  - 支持7天/30天/90天時間範圍切換
- **風控看板**：
  - 大額交易實時預警
  - 異常登錄檢測
  - 支付失敗率監控
  - 系統健康度評估
- **訂單管理**：
  - 訂單全鏈路追蹤
  - 搜索和篩選功能
  - CSV數據導出
- **自動化運維**：每5分鐘自動刷新數據
